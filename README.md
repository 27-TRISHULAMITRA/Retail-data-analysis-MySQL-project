# Retail Store Management System – MySQL Mini Project

## 📌 Project Overview
This project is a *Retail Store Management System* developed using *MySQL*.  
It is designed to manage and analyze retail store operations such as:

- Customer management
- Product inventory
- Order processing
- Sales tracking
- Data analysis using SQL queries

The project demonstrates practical usage of:
- Database design
- Table relationships
- SQL queries
- Joins
- Aggregate functions
- Subqueries
- Views and advanced SQL concepts

---

## 🛠️ Technologies Used

- *MySQL*
- SQL Queries
- Relational Database Management System (RDBMS)

---

## 🗂️ Database Structure

The database includes the following main tables:

### 1. Customers
Stores customer information such as:
- Customer ID
- Name
- Email
- Phone Number
- Account Creation Date

### 2. Products
Stores product details:
- Product ID
- Product Name
- Category
- Price
- Stock Quantity

### 3. Orders
Stores order-related information:
- Order ID
- Customer ID
- Order Date
- Order Status
- Total Amount

### 4. Order Details (if included in schema)
Stores:
- Product-wise order information
- Quantity
- Price per item
